Aplikasi berbasis Web sederhana yang menggunakan LangChain, Groq (Llama 3.3), dan Streamlit untuk meringkas konten dari video YouTube atau artikel website secara otomatis. 
Aplikasi ini mendukung ringkasan dalam bahasa indonesia maupun inggris sesuai pilihan pengguna

**Fitur Utama**
- YouTube Summarizer: Mengambil transkrip video YouTube secara otomatis (mendukung bahasa Indonesia dan Inggris).
- Website Summarizer: Mengekstrak teks dari artikel atau laman web menggunakan UnstructuredURLLoader.
- Multi-language Output: Hasil ringkasan dapat disesuaikan ke dalam bahasa target (Indonesia atau Inggris).
- Fast Inference: Menggunakan model Open Source Llama 3.3-70b di infrastruktur Groq untuk kecepatan tinggi.

- **Teknologi yang digunakan**
- - Framework : Streamlit
  - Orchestration : Langchain
  - LLM : GROQ (Model : llama-3.3-70b-versatile)
  - LoaderL YoutubeLoader & UnstructuredURLLoader
